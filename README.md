# PlantUML Editor

A web-based PlantUML editor that allows you to code and render PlantUML diagrams directly in your browser. No installation required.

**Live:** https://plantuml.aristorinjuang.com

## Features

- Write PlantUML code and see real-time diagram rendering
- Auto-save functionality - your work is saved automatically
- File management - create snapshots and load saved diagrams
- Responsive layout with mobile-friendly tabs
- Dark/light theme toggle
- Keyboard shortcuts (Ctrl+S to save, Ctrl+O to open)

## Tech Stack

- **Vite** - Fast build tool
- **Tailwind CSS v4** - Utility-first CSS framework
- Vanilla JavaScript - No framework dependencies

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
